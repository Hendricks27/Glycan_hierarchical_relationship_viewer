How to use the lib

1. First put 3 script tags for this JavaScript widget, d3 library and vis v4.19.1 (Please use v4.19.1) in head part of your HTML.
For the d3 and vis, I recommend you just use script from CDN, so you don't necessarily need to save a local copy.
Put a style tag for vis in the head part of your HTML.

2. Create a empty div with id. This is the place where I will put my widget in.

3. Create a parameter JavaScript Object, and pass it to glycanview_new.init(para_obj);

Here is the simplified demo. Only contains the necessary part.<-If you are using Chrome, press Ctrl+u to see the source code

What can you do with the lib

1. View the relationship among a series of glycans

2. Interact with interface, such as drag the node, zoom-in&out, add edit and delete the images and edges(Coming out soon)
